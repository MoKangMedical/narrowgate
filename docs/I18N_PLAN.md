## 二、技术方案

### 推荐方案：JavaScript动态切换
**优势：**
- 保持单文件结构
- 实现简单，易于维护
- 用户体验好（无需刷新页面）
- 保存用户语言偏好

**实现方式：**
1. 使用 `data-i18n` 属性标记可翻译文本
2. 创建语言配置文件（JSON格式）
3. 实现语言切换函数
4. 保存用户偏好到 localStorage

### 替代方案对比
| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| A. 双HTML文件 | 实现简单 | 维护困难，代码重复 | ⭐⭐ |
| B. JS动态切换 | 单文件，易维护 | 需要重构代码 | ⭐⭐⭐⭐⭐ |
| C. i18n框架 | 功能强大 | 复杂，增加依赖 | ⭐⭐⭐ |

### JavaScript实现
```javascript
// 语言配置
const i18n = {
  zh: { /* 中文配置 */ },
  en: { /* 英文配置 */ }
};

// 当前语言
let currentLang = localStorage.getItem('lang') || 'zh';

// 切换语言
function switchLanguage(lang) {
  currentLang = lang;
  localStorage.setItem('lang', lang);
  updatePageContent();
  updateLangButton();
}

// 更新页面内容
function updatePageContent() {
  const elements = document.querySelectorAll('[data-i18n]');
  elements.forEach(el => {
    const key = el.getAttribute('data-i18n');
    el.textContent = i18n[currentLang][key];
  });
}

// 更新语言按钮
function updateLangButton() {
  const btn = document.getElementById('lang-switch');
  btn.textContent = currentLang === 'zh' ? 'EN' : '中';
}

// 页面加载时初始化
document.addEventListener('DOMContentLoaded', () => {
  updatePageContent();
  updateLangButton();
});
```
