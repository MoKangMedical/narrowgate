# 窄门理论体系 — NarrowGate Theory

> **"引到永生，那门是窄的，路是小的，找着的人也少。"**
> 本文档是窄门平台的理论根基、方法学体系和培训框架。

---

## 一、窄门核心命题

### 1.1 一句话定义

**窄门 = 你最不想面对但最需要面对的那件事。**

### 1.2 窄门公式

```
窄门优先级 = 痛苦回避度 × 成长杠杆率
```

- **痛苦回避度**（0-100）：你有多不想面对这件事
- **成长杠杆率**（0-100）：穿越它后能带来多大的改变
- 两个维度相乘，得分最高的就是你最需要穿越的窄门

### 1.3 宽门 vs 窄门

| 维度 | 宽门 | 窄门 |
|------|------|------|
| 路径 | 容易、多数人走 | 艰难、少数人走 |
| 短期感受 | 舒服 | 痛苦 |
| 长期结果 | 停滞、后悔 | 成长、释放 |
| 本质 | 回避真实自我 | 面对真实自我 |
| 比例 | 95%的人 | 5%的人 |

## 二、灵魂审计方法论

### 2.1 什么是灵魂审计？

灵魂审计 = 苏格拉底式追问 + AI回避检测 + 鉴别诊断式排查

不是心理咨询（不治疗），不是教练辅导（不指导），是**帮你看见你不想看见的东西**。

### 2.2 五维度模型

| 维度 | 探索内容 | 核心问题 | 对应大师 |
|------|----------|----------|----------|
| 🔮 认知 | 信念、思维模式、自欺 | 你相信什么是真的？它真的真吗？ | 苏格拉底 |
| 🌑 情绪 | 情感、压抑、恐惧、阴影 | 你在感受什么？你害怕什么？ | 荣格 |
| ⚡ 行为 | 习惯、执行、拖延 | 你在做什么？你没做什么？ | 清醒者 |
| 🪞 关系 | 人际、边界、讨好、投射 | 你在关系中是谁？你害怕被看见什么？ | 镜像者 |
| 🏗️ 事业 | 目标、系统、杠杆 | 你的计划是计划还是幻想？ | 架构师 |

### 2.3 回避检测机制

系统实时检测五种回避模式：

| 回避类型 | 信号词 | 背后的真相 |
|----------|--------|-----------|
| **否认** | "没有"、"不是"、"还好" | 不愿承认问题存在 |
| **转移** | "说起来"、"对了"、"另外" | 话题太痛，想跑 |
| **最小化** | "一点点"、"稍微"、"可能" | 承认问题但不想面对严重性 |
| **合理化** | "因为"、"没办法"、"大家都" | 给问题找借口 |
| **攻击** | "你凭什么"、"你懂什么" | 被戳中要害的防御反应 |

### 2.4 四层追问深度

```
L1 表面层 → L2 中间层 → L3 深层 → L4 核心层
"你做了什么"  "为什么做"  "你害怕什么"  "你是谁"
```

- **L1 表面**：事实层面，你在做什么/不做什么
- **L2 中间**：动机层面，为什么这样做
- **L3 深层**：恐惧层面，你真正害怕的是什么
- **L4 核心**：身份层面，这反映了你是谁

大多数人在L1就停了。窄门帮你到达L4。

## 三、进化金字塔 — 五层成长模型

### 3.1 架构

```
          ✨  第5层：神性层 — 存在即穿越
        ┌───┐
        │ 4 │  第4层：精通层 — 穿越成为习惯
      ┌─┴───┴─┐
      │   3   │  第3层：突破层 — 第一次穿越
    ┌─┴───────┴─┐
    │     2     │  第2层：觉醒层 — 看见枷锁
  ┌─┴───────────┴─┐
  │       1       │  第1层：睡眠层 — 不知道自己在睡
└───────────────────┘
```

### 3.2 每层详解

#### 😴 第1层：睡眠层（Sleep）

**特征：**
- "我就是这样的人"
- 不觉得有什么需要改变
- 把问题归因于外部
- 舒适区就是整个世界

**窄门信号：** 你开始怀疑"也许问题在我"的那一刻

**经验值门槛：** 0（起点）

---

#### 👁️ 第2层：觉醒层（Awakening）

**特征：**
- 开始问"为什么我总是这样"
- 对自己的行为模式有觉察
- 感到不适和焦虑
- 想要改变但不知道如何

**窄门信号：** 当你开始用行动而不是觉察来回应的时候

**经验值门槛：** 100

---

#### ⚡ 第3层：突破层（Breakthrough）

**特征：**
- 完成了第一次穿越挑战
- 体验到"另一种可能"
- 开始建立新习惯
- 旧模式开始松动

**窄门信号：** 当突破从一次性事件变成稳定习惯的时候

**经验值门槛：** 300

---

#### 🎯 第4层：精通层（Mastery）

**特征：**
- 穿越成为习惯
- 能够指导他人穿越
- 看到更深层的枷锁
- 稳定的新模式

**窄门信号：** 当窄门变成你的路，不再需要刻意穿越的时候

**经验值门槛：** 600

---

#### ✨ 第5层：神性层（Divinity）

**特征：**
- 窄门成为常态
- 不再需要刻意努力
- 存在即穿越
- 你的光芒照亮他人的窄门

**窄门信号：** 你已经不需要证明了

**经验值门槛：** 1000

|------|--------|------|
| 每日挑战完成 | +10 | 稳定积累 |

## 四、穿越训练系统 — 30天方法学

### 4.1 总体框架

穿越训练 = 30天 × 每日挑战 × 渐进难度 × 见证人验证

```
第1周：面对恐惧（承认 → 第一次尝试）
  ↓
第2周：建立节奏（每日挑战 → 形成习惯）
  ↓
第3周：突破瓶颈（难度升级 → 突破舒适区）
  ↓
第4周：巩固穿越（从刻意到自然 → 内化新身份）
```

### 4.2 每日挑战结构

每个挑战包含四个部分：

```
1. 🎯 今日行动 — 具体做什么（可执行、可验证）
2. 😰 预期恐惧 — 做之前你害怕什么
3. 📸 完成证据 — 截图/文字/见证人确认
4. 💭 穿越感悟 — 做完后发现什么（恐惧 vs 现实）
```

### 4.3 挑战难度曲线

| 阶段 | 天数 | 难度 | 目标 |
|------|------|------|------|
| 热身 | D1-D3 | 3/10 | 建立行动感 |
| 基础 | D4-D7 | 5/10 | 面对表层恐惧 |
| 进阶 | D8-D14 | 6/10 | 建立每日节奏 |
| 深水 | D15-D21 | 7/10 | 突破核心恐惧 |
| 穿越 | D22-D28 | 8/10 | 整合新身份 |
| 收尾 | D29-D30 | 回顾 | 总结穿越旅程 |

## 五、大师引导者体系

### 5.1 设计哲学

**好的AI不应有问必答，而应像苏格拉底一样追问引导。**

- 不给答案，给问题
- 不给安慰，给挑战
- 不给依赖，给成长

### 5.2 七位大师

| 大师 | 角色 | 维度 | 解锁层级 | 一句话 |
|------|------|------|---------|--------|
| 🦉 苏格拉底 | 追问者 | 认知 | L1+ | 用问题瓦解你的确定性 |
| 🌑 荣格 | 阴影猎手 | 情绪 | L1+ | 你没有面对的会变成你的命运 |
| ⚡ 清醒者 | 执行官 | 行为 | L1+ | 我不关心你知道什么，只关心你做了什么 |
| 🪞 镜像者 | 关系分析师 | 关系 | L1+ | 你在他人身上看到的都是你自己 |
| 🏗️ 架构师 | 系统思维者 | 事业 | L2+ | 你不是棋子，你是下棋的人 |
| 🔮 炼金术士 | 变形者 | 认知 | L3+ | 在最深的黑暗中找到最亮的光 |
| 🚪 守门人 | 窄门本身 | 全部 | L4+ | 我检验你是否配得上 |

### 5.3 大师对话机制

每位大师通过MIMO AI实时生成追问，遵循以下规则：

1. **不接受表面答案** — 回答太浅会触发追问
2. **检测回避模式** — 否认/转移/最小化/合理化会触发提醒
3. **层层深入** — 每轮对话推向更深层
4. **保持角色** — 每位大师有自己的性格、哲学和语言风格

## 六、见证人网络

### 6.1 为什么需要见证人？

**个人成长的最大敌人不是缺乏知识，是缺乏外部压力。**

- 你可以对自己撒谎
- 你无法对看你穿越的人撒谎
- 见证人 = 社交压力 = 行动的催化剂

### 6.2 见证人类型

| 类型 | 角色 | 适用场景 |
|------|------|---------|
| 朋友 | 平等支持 | 日常挑战 |
| 家人 | 深度了解 | 关系窄门 |
| 导师 | 专业指导 | 事业窄门 |
| 同伴穿越者 | 互相见证 | 所有窄门 |

### 6.3 见证机制

1. **添加见证人** → 见证人收到通知
2. **每日挑战** → 见证人看到你的进度
3. **穿越验证** → 见证人确认你真的做了
4. **里程碑** → 见证人在你突破时见证

## 九、理论宪法关联

窄门遵循莫康医学理论宪法（THEORETICAL_CONSTITUTION.md）：

- **认知论**：鉴别诊断式决策 → 灵魂审计的五维度排查
- **方法论**：Harness理论 → 思想Harness（引导者+进化金字塔+回避检测）
- **价值论**：红杉论点 → 卖思想成长结果（困惑→洞见→行动），非卖对话工具
- **价值论**：思想基础设施理论 → 窄门是Level 4（思想基础设施），构建思维方式

**窄门的Harness类型：思想Harness**
- 引擎：灵魂审计 + 穿越训练 + 大师引导
- 护城河：理论体系 + 7位大师人格 + 回避检测机制 + 进化金字塔
- 非技术壁垒，而是思想壁垒——代码可以抄，哲学体系抄不了
