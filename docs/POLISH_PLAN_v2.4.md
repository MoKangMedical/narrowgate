# 窄门首页打磨计划 v2.4

> 目标：按国际品牌10分标准打磨首页（index.html）
> 参考品牌：Apple, Aesop, Headspace, Calm
> 开始时间：2026-04-19

## 设计原则

1. **克制 > 丰富** — 少即是多
2. **大量留白** — 呼吸感
3. **精确的字体层级** — 清晰的信息架构
4. **微动效** — 不炫技，提升体验
5. **温暖但不幼稚** — 哲学感
6. **哲学感但不故弄玄虚** — 真诚

## 问题诊断

### 当前问题
1. **Hero区域** — 间距不够，视觉冲击力不足
2. **核心命题** — 表达过于直白，缺乏诗意
3. **五层进化** — 金字塔视觉可以更精致
4. **灵魂审计** — 对话区域可以更沉浸
5. **大师引导者** — 卡片设计可以更高级
6. **30天训练** — 信息密度太高
7. **课程界面** — 与其他部分风格不统一
8. **关于部分** — 可以更精简

### 代码问题
- 360行和362行有重复的 `let selectedMasterId = "";`
- CSS类名不统一（有些用bg-[#xxx]，有些用自定义类）
