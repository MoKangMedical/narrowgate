## 四、课程与经验值对应

### 4.1 每日经验值分布

| 日 | 经验值 | 理由 |
|----|--------|------|
| D1-D6 | 10 | 基础行动，建立节奏 |
| D7 | 20 | 第一周总结，反思深化 |
| D8-D13 | 10-15 | 难度提升，建立系统 |
| D14 | 20 | 第二周总结 |
| D15-D20 | 15-20 | 深度探索，突破挑战 |
| D21 | 25 | 第三周总结 |
| D22-D28 | 15-20 | 身份整合，系统建立 |
| D29 | 25 | 穿越准备 |
| D30 | 50 | 毕业穿越，重大突破 |

### 4.2 总经验值计算

```
基础课程完成: 30天 × 平均15XP = 450XP
每周总结奖励: 3次 × 20-25XP = 65XP
毕业穿越奖励: +200XP
见证人验证: +20XP × 验证次数
突破记录: +50XP × 突破次数

预估总经验值: 735-800+ XP
对应进化层级: L3 突破层
```
